# 🏺 Tablas - Sistem Tabungan Kelas

[![Laravel](https://img.shields.io/badge/Laravel-11.x-FF2D20?style=for-the-badge&logo=laravel)](https://laravel.com)
[![Vue](https://img.shields.io/badge/Vue.js-3.x-4FC08D?style=for-the-badge&logo=vue.js)](https://vuejs.org)
[![TailwindCSS](https://img.shields.io/badge/Tailwind-3.x-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com)

**Tablas** adalah aplikasi manajemen tabungan kelas yang dirancang untuk transparansi keuangan antara bendahara dan siswa. Dibangun menggunakan stack modern **Laravel (Inertia.js)** dan **Vue.js**.

## ✨ Fitur Utama

-   **Dashboard Statistik**: Ringkasan total kas, jumlah siswa, dan transaksi harian.
-   **Cek Saldo Mandiri**: Siswa dapat mengecek saldo secara real-time hanya dengan memasukkan NIS di landing page.
-   **Manajemen Transaksi**: Pencatatan setoran (deposit) dan penarikan (withdrawal) yang aman dengan *database transaction*.
-   **Database Auto-Sync**: Saldo siswa otomatis terupdate setiap kali transaksi dicatat.
-   **UI Modern**: Antarmuka profesional dengan Tailwind CSS dan efek Glassmorphism.

## 🚀 Teknologi yang Digunakan

-   **Backend**: Laravel 11
-   **Frontend**: Vue.js 3 (Inertia.js)
-   **Styling**: Tailwind CSS
-   **Database**: MySQL
-   **Icons**: Heroicons
